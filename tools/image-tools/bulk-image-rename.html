Why bother renaming in bulk?

A camera or phone hands you filenames like IMG_2041.jpg — useless once you're sorting hundreds of them. This tool lets you swap an entire batch to something meaningful — a project name plus a running number — in one pass instead of clicking rename on each file.

It runs entirely in your browser, works the same on a laptop or a phone, and finishes with your organized files bundled into one ZIP so there's nothing left to clean up afterward.

How it works

  • Load one photo or a whole batch
  • Type the base name you want them to share
  • Optionally turn on prefix, suffix, or a starting number
  • Check the live preview of each new filename
  • Download one by one, or everything as a ZIP
