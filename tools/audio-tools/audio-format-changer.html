MP3 to WAV, WAV to MP3, M4A to OGG — one tool, no signup

Your file is decoded into raw PCM samples by the browser's Web Audio API, then re-encoded into the format you pick — MP3, WAV, or OGG — entirely inside the tab you're already in. There's no account to create, no watermark stamped onto the result, and no file size cap imposed by us. The only limit is how much memory your device can spare for the decoded buffer.

MP3 output uses the lamejs encoder running in JavaScript, so the bitrate you pick (128 / 192 / 256 / 320 kps) directly controls how the file balances size against how close it stays to the original sound. WAV output is lossless PCM, so it preserves the decoded waveform exactly — at the cost of a much larger file. OGG output uses your browser's built-in MediaRecorder, so it works in Firefox and Chrome but may not be supported in every browser.

Why convert audio in the browser instead of using a website that uploads your file?

An audio converter that runs entirely in your browser has a different threat model than the typical online audio converter. The upload-based kind sends your recording to a server you don't control, processes it there, then hands you a download link. Browser-side conversion never sends anything anywhere — the file is opened locally, decoded by the Web Audio API on your own device, re-encoded by JavaScript (or by your browser's own MediaRecorder), and returned to you directly as a download. That matters more for audio than for images: recordings often contain voice memos, meetings, calls, or music you simply don't want sitting on a third-party server.

Audio formats this tool supports — and when to pick which

Four output formats are supported, each chosen for a real use case rather than for a marketing checkbox. Pick the one that matches what you'll do with the converted file:

MP3 — the universal default. Plays everywhere, small per minute of audio, and the 128 / 192 / 256 / 320 kbps bitrate control is enough to balance size against transparency. Pick MP3 when you just need the file to play on any device.

WAV (lossless) — uncompressed PCM. Converting to WAV loses zero audio quality, because WAV stores the raw waveform. Pick WAV when you're archiving a recording, editing further in another tool, or handing the file to a video editor.

M4A / AAC — Apple's audio container, smaller than MP3 at matching quality. Most free online converters gate it behind a paid tier. M4A output here runs through your browser's MediaRecorder, so Chromium-based browsers (Chrome, Edge, Brave, Vivaldi, Opera) can produce it; Firefox and Safari show a clear "not supported" indicator.

OGG / Opus — open, royalty-free, and the format most online converters skip. Opus inside an OGG container is what Firefox and Chrome use internally for speech and music; for the same audible quality it tends to be smaller than MP3. Pick OGG when you want an open format, or when you're serving audio to a website that supports it.
