How the rotation actually works

Every page in a PDF carries a rotation property that tells a reader which way up to display it. This tool reads that property, adds your chosen angle to it, and writes the file back out — the underlying content never gets redrawn or re-rendered, so there's no quality loss at all.

Because rotation is a standard PDF page property, it's respected by every compliant reader — Adobe Acrobat, browser PDF viewers, and everything in between.
