How hitting a target size actually works

Every page is rendered onto a canvas, then re-encoded as a JPEG image at a quality level chosen by a search — trying progressively lower or higher quality until the rebuilt PDF lands as close as possible to your target size. If your target can't realistically be reached, the closest achievable size is returned instead of a broken or unreadable file.

Because every page becomes an image, this is best for sharing or uploading where an exact size limit matters more than being able to select or search the text afterward.

When this is worth doing

Upload size limits: forms, portals, and email attachments that reject files over a set size.
Consistent batch sizing: get a whole folder of PDFs under the same cap.
Bandwidth-conscious sharing: keep files small for slow connections.
Storage limits: fit more files into a fixed quota.
