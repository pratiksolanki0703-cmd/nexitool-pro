How the password protection actually works

This tool implements the PDF Standard Security Handler — the same encryption mechanism defined in the PDF specification itself. Your password derives an encryption key which locks every string and stream inside the document with 128-bit RC4 encryption, then the file is rebuilt with an encryption dictionary that tells any PDF reader to prompt for that password before opening it.

Because it's the standard mechanism baked into the PDF format, the locked file opens correctly in Adobe Acrobat, browser PDF viewers, and virtually any other compliant reader — not just this site.
