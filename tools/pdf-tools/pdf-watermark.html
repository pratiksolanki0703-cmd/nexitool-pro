How the watermarking actually works

This tool embeds a bold font directly into the PDF and draws your chosen text diagonally across the center of every page, at the opacity and size you set. The text becomes part of the page content itself, so it stays in place if the file is printed or converted.

Everything happens through pdf-lib running in your browser tab — the file is never sent to a server.
