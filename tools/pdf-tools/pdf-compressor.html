What actually makes a PDF smaller here

A lot of a PDF's weight sits in metadata you never see — title, author, software version, revision history — plus a loosely-packed internal object structure. This tool clears that metadata out and rebuilds the structure more tightly, which is usually enough to noticeably shrink a text-heavy document.

It doesn't yet re-encode images embedded inside the PDF, so a scanned document or an image-heavy report won't shrink nearly as much as a report full of plain text — that's a real limitation worth knowing about upfront, not a bug.
