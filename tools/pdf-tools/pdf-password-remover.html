How the password removal actually works

A password-protected PDF stores an encryption dictionary describing how its file key was derived. This tool re-derives that key from the password you enter, verifies it against the file's stored check value, and only then decrypts every object and rewrites the file without the encryption dictionary.

If the password doesn't match, the file is rejected before anything is decrypted — this tool unlocks files you can already open, it does not crack unknown passwords.
