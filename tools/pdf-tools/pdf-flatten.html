How PDF flattening actually works

A fillable PDF form stores its text boxes, checkboxes, and radio buttons as interactive field objects layered over the page. This tool takes the current value of every field, draws it permanently into the page content, and removes the interactive field layer — so the result looks identical but can no longer be edited.

Files with no form fields at all are detected and skipped, since there's nothing to flatten.
