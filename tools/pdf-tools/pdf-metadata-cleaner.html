How the metadata reading and cleaning actually works

Every PDF carries a document info dictionary with fields like Title, Author, Subject, Keywords, Creator, and Producer — often filled in automatically by the software that created the file. This tool reads those fields the moment you add a file, and displays them right on its card.

Clicking Clean sets every field to empty and rebuilds the file, so the exported PDF carries no trace of who made it or what software touched it.
